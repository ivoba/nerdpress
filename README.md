# nerdpress

[Nerdpress.org](http://nerdpress.org) built on [Bedrock](https://roots.io/bedrock/)

## Install

copy env file, adjust if neccesary and add keys

    cp .env.example .env
    
    docker-compose up

    docker-compose run php composer install

## Use

    http://localhost:8088
    http://localhost:8088/wp/wp-admin
